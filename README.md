🔹 学习 & 记忆小技巧

平假名：用于日语本身（语法、词尾）

片假名：外来语 / 人名 / 强调

を：读 o（不是 wo）

ふ：介于 fu / hu 之间

ん：不是 ni，是鼻音 n
